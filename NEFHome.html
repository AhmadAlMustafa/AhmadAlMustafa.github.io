Our system have hundreds of features that could be used and I won't be able to cover all features but I will try to explain some of important features.

Users, roles and policies

NEF financial system is a system that manipulates in many countries (Syria-Jordan-Iraq-Sudan) with many branches and services. More than 3,292 borrowers, 16,167 learners, 316 training and coaching mdules.

So it has a very strong and secured users,roles and policies system that enables admins to define users, roles, policies and relate them to different centers, donors and accessibilities.

The administrator also has the ability to enable or disable notifications for specific users in our system's notification system using emails or whatsApp messages, the notifications include information about the System Transactions and money movements.

Flexible picklist's definitions

NEF System enables users to define multi types of picklist like type of business, holidies, nationalities and much more.

this feature gives users the flxiblility to customize configuration for each center using system parameter page.

Flexible workflow's steps

NEF provide flexible way to define loan's workflow steps so the user can define steps then create worflows for each type of loan based on loan amount,center or donor with easy drag and drop popup, So when the user creates a new loan it will follow the appropirate workflow going from step to other based on the decision taked by the step's employee .

At each step, the employee has the ability to redirect the loan to the next step or return it to the previous step while adding his notes to the next step employee.

Cashboxes

The user can define multi cashboxes for disbrsement or repayment then he can easily monitor all transactions, money movements, cash balances and export reports as pdf or excel. also each cashbox have End-Of-Day report that should be uploaded at the end of day to close cashbox or a notification email will be sent to lending managers notify them that the cashbox still opened.

Report builder

One of the coolest feature in our system is the report builder feature, this feature enables the admin to create report based on custom entities,columns and filters.

For example we can create second-quarter-report to get applicationCode, requested loan amount and client first name for all loans created after 1-Jan-2023 and have amount > 400. then we could use this report later at anytime
